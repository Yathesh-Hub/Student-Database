# Student Details Management System

This is a simple CRUD backend mini project built using Node.js, Express, and MongoDB.
It helps to store and manage student details using REST APIs.

## 📌 What this project does
- Add student details
- Search student by ID
- View all students
- Update student details
- Delete student records

## 🛠️ Technologies Used
- Node.js
- Express.js
- MongoDB
- Mongoose
- Postman (for testing APIs)
